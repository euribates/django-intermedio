## Descripción del curso

Proyecto a realizar durante el curso: **spidercheck**

Temas que se van a trabajar:

- *Authentication*
- *Context Processors**
- *Custom tags and filters*
- *Databases*
- *Deployment on production*
- *Generated fields*
- *Logging*
- *Management commands*
- *Middleware*
- *Migrations*
- *Objects managers*
- *Path converters*
- *Third-part apps and modules*
